# EasyAdmin(极易后台开发框架)

>thinkPHP 6.0+AdminLTE 开发的后台开发框架

> 运行环境要求PHP7.1+。

## 主要新特性

* 采用thinkPHP 6.0+AdminLTE 开发
* 参考了fastadmin，初期可看作fa的tp6.0升级版

## 安装

~~~
composer create-project topthink/think tp 6.0.*-dev
~~~

如果需要更新框架使用
~~~
composer update topthink/framework
~~~
